## 관리 원칙

- 공개 문서는 제품과 코드 구조를 이해하는 데 필요한 내용만 담습니다.
- 실제 secret, 운영 로그, DB dump, 임시 테스트 자료는 커밋하지 않습니다.
- 배포 절차와 운영 인수인계 문서는 로컬 전용 문서로 관리합니다.
