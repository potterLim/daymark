Password recovery

로그인 흐름은 가볍게 유지하고, 복구는 확실하게 준비했습니다.

가입 시 입력한 이메일이 인증된 상태라면 비밀번호 재설정 링크를 바로 보낼 수 있습니다. 아직 인증되지 않았다면 같은 흐름 안에서 이메일 소유 확인 링크를 다시 안내합니다.

Secure link 재설정 링크는 한 번만 사용할 수 있습니다.

만료 시간이 지나거나 이미 사용된 링크는 다시 사용할 수 없습니다.

Generic response 입력 결과는 계정 존재 여부를 노출하지 않습니다.

보안과 사용성을 함께 고려하면서도, 인증 전 계정은 복구 경로를 복원하도록 설계했습니다.
